CSS Shape-Outside - Circle

Demonstration of the traditional way text is wrapped around an image. The image to the left is a circle but, because of the box model, text is wrapped around it with square sides. The text to be wrapped was bound by this square because css is based off the box model. If you wanted to wrap text around the image, you were limited to this square design.
